## Banner Optimization

### Technical Specifications (2026)

- **Dimensions**: 1500 x 500 pixels (3:1 aspect ratio)
- **File Size**: Under 2-5 MB
- **Formats**: JPG, PNG (GIF not recommended, no animation support)
- **Safe Area**: 1500 x 360 pixels (account for profile picture overlap bottom left + cropping)
- **Key Content Placement**: Center (sides may crop on mobile)
- **Typography**: Minimum 24px for mobile, 48-72px for headlines, 24-36px for subheadings
- **Profile Picture Overlap**: Leave ~200px bottom left clear

### Design Strategy

**Impact**: Well-designed banner can boost profile visits by **up to 30%** and improve follower conversion rate

**Content Options**:
1. **Value Proposition Visual** â€” "AI for Call Centers: 20% CSAT Increase" + visual elements
2. **Social Proof** â€” "7 Years Voice AI Production | 500K+ Interactions Analyzed"
3. **Build in Public** â€” "160+ Autonomous PRs | Building AI Agents in Public" + GitHub stats visual
4. **Company + Personal** â€” Ender Turing branding + personal tagline
5. **Clean/Minimal** â€” Simple gradient or abstract tech visual (no text, relies on bio)

**Best Practices**:
- Simple and clean (not cluttered)
- High-quality images (minimum 1500x500px)
- Emotional appeal (tech aesthetic, credibility, innovation)
- Regular updates (align with campaigns, launches, milestones)
- Mobile preview (test before publishing)

**Recommendation**: Start with simple text-based banner showing value prop + credibility, refresh after Premium activation.

### What Works
1. **Clear value proposition** â€” visitors know immediately why to follow

3. **Credibility markers** â€” 7 years, 500K interactions, @EnderTuring, 160+ PRs
