## How to Find This Data in X Analytics

1. Go to https://analytics.x.com
2. Click "Tweets" tab for tweet-level data
3. Adjust date range to last 7 days
4. Sort by impressions or engagements
5. Note top performers

For follower data:
1. Visit your profile page
2. Check follower count (compare to last week's report)

For engagement rate:
1. Divide total engagements by total impressions
2. Multiply by 100 for percentage
