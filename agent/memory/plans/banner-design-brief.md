## Design Direction

### Visual Style Options

#### Option 1: Clean Tech Professional (RECOMMENDED)
- **Background**: Dark gradient (navy → deep purple) OR solid dark color
- **Text**: White, bold, modern sans-serif (Helvetica, Inter, Poppins)
- **Accent**: Neon blue or cyan highlights (tech feel)
- **Graphics**: Subtle AI/tech patterns (circuit lines, neural network nodes)
- **Mood**: Professional, innovative, technical credibility

**Why**: Tech audience expects polished, modern design. Dark background with bright text = high contrast = mobile-readable.

#### Option 2: Data Visualization
- **Background**: Abstract waveform or audio spectrum visualization
- **Text**: Overlaid on translucent panels for readability
- **Theme**: Call center focus (voice/audio theme)
- **Color**: Teal/cyan (associates with tech/analytics)
- **Mood**: Data-driven, analytical, domain expertise

**Why**: Reinforces call center AI niche with relevant visual metaphor.

#### Option 3: Minimalist Statement
- **Background**: Single solid color (black, navy, or white)
- **Text**: Large, bold typography (statement text only)
- **Graphics**: None or minimal geometric shapes
- **Mood**: Confident, bold, lets the words speak

**Why**: Easier to DIY in Canva, works on all devices, timeless.

## Typography Rules

### Font Recommendations
- **Primary text (headline)**: Bold, 60-80pt
- **Secondary text (subheadline)**: Medium weight, 40-50pt
- **Tertiary text (details)**: Regular, 30-40pt

### Font Choices
**Professional/Tech**:
- Inter (modern, readable)
- Poppins (friendly, bold)
- Helvetica Neue (classic, clean)
- Montserrat (geometric, strong)

**If DIY in Canva**: Use built-in fonts like "Anton" (bold headlines), "Roboto" (body), or "Oswald" (tech feel)

### Text Placement
- Keep critical text in center 1200x300 safe zone
- Avoid text behind profile picture area (bottom left)
- Use high contrast (white text on dark BG, or dark text on light BG)
- Minimum 12px spacing from edges

## DIY Instructions (Using Canva)

### Step-by-Step for Non-Designers

1. **Go to Canva.com** → Sign up (free account works)
2. **Create custom dimensions**: Click "Custom size" → 1500 x 500 px
3. **Choose template**: Search "Twitter header" → browse templates
4. **Customize**:
   - Replace text with Version A content (see above)
   - Change background color to dark navy (`#0A1929`)
   - Change text color to white
   - Adjust font to "Poppins Bold" or "Montserrat Bold"
   - Add subtle graphics (search "circuit" or "tech pattern" in elements)
5. **Position text**:
   - Leave 200px clear on bottom left (profile pic area)
   - Center main content horizontally
6. **Download**: Click "Share" → "Download" → PNG format
7. **Upload to X**: Profile → Edit profile → Header photo → Upload

**Time estimate**: 15-30 minutes (first time)
