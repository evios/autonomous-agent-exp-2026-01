### 2. CTA Discipline
- **Rule**: Every post >50 impressions includes soft CTA
- **Evidence**: Rowan Cheung — first 55K newsletter subs = 100% organic from X CTAs
- **Templates**: "Building in public → [repo]", "Weekly retros → follow"
- **Timing**: Don't wait for 10K followers, CTA from Day 1

### 3. Educational Simplification
- **Template**: "Here's [complex concept]. Plain English: [1-2 sentences]. Why it matters: [implication]."
- **Evidence**: Rowan Cheung (0 → 300K in 4 months) — "Simplify complex → easily-digestible"
- **Use for**: Specification Engineering, PDCA cycles, queue discipline, cognitive debt

### 4. Platform Specialization
- **Principle**: X for hooks, depth elsewhere
- **Evidence**: Andrew Ng — X for concise, LinkedIn for long-form
- **Application**: X posts = 1-3 sentences + CTA, depth in repo/LinkedIn/Gists
- **Why**: X algorithm rewards brevity (our data: long posts <10 imp avg)
